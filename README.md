# TransitOps

TransitOps is a transit and fleet management application currently under development.

The goal is to build a simple, reliable platform for managing vehicles, drivers, routes, trips, and day-to-day transport operations.

## Tech Stack

* Next.js
* Express.js
* PostgreSQL
* Prisma
* TypeScript
* Tailwind CSS

## Project Structure

```text
transitops/
├── client/    # Next.js frontend
├── server/    # Express backend
└── README.md
```
